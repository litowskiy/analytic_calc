#Аналитический калькулятор
##Функционал
- Поддерживает базовые функции калькулятора: +, -, *, /, ^
- Поддерживает тригонометрические функции: sin, cos, tan, cot
- Поддерживает другие аналитические функции и константы: log, pi, E, exp
- Умеет раскрывать скобки и упрощать выражения
- Умеет брать производные

##Режимы работы
- "=" - вычисляет значение выражения. **Пример**: 2+2 -> 4, sin(30) -> 1/2, x+1 -> останется x+1  
- "d/dx" - берет производную. **Пример**: sin(x) -> cos(x), x^2 -> 2*x  
- "упростить" (simplify) - упрощает выражение. **Пример**: (x^2 - 1)/(x - 1) -> x + 1, sin(x)^2 + cos(x)^2 -> 1  
- "раскрыть" (expand) - раскрывает скобки и степени. **Пример**: (x+2)^2 -> x^2 + 4*x + 4  
- "разложить" (factor) - раскладывает на множители. **Пример**: x^2 - y^2 -> (x - y)*(x + y)  
- "очистить" (clear) - очищает поле ввода и результат
